DESEMPENHO:

Throughput (largura de banda): Mede a quantidade de tarefas executada em uma unidade de tempo. Servidores procuram aumentar esta característica.

Tempo de resposta (tempo de execução): tempo necessário para o computador executar uma tarefa, incluindo acessos à memória, E/S, etc.

OBS: em um sistema complexo, aumentar o tempo de resposta contribuirá para um aumento da largura de banda, o contrário também é verdadeiro.

desempenho_x = 1 / tempo_execucao_x

n =  desempenho_x / desempenho_y

OBS: se n é maior que 1, dizemos que "x é n vezes mais rápido do que y".

Tempo de execução de CPU (tempo de CPU): quanto tempo uma CPU gasta para executar uma única tarefa.

Tempo de CPU do usuário: o tempo de CPU gasto em um programa.

Tempo de CPU do sistema: tempo de CPU gasto no SO realizando tarefas em favor do programa.

Tempo de execução da CPU para um programa = quant ciclos de clock x tempo do ciclo de clock

CPI = clock por instrução. É o valor médio de ciclos que o programa leva para executar cada instrução do programa

Taxa de clock: frequência de clock/quantos ciclos de clock por segundo

Contador de instruções: número de instruções executadas pelo programa

Equação clássica de desempenho de CPU: Tempo de CPU = (Contador de instrução x CPI) / taxa de clock; 

Determinada aplicação escrita em Java roda por 15 segundos em um
processador de desktop. Um novo compilador Java é lançado, exigindo apenas
60% das instruções do compilador antigo. Infelizmente, isso aumenta o CPI
em 1,1. Com que velocidade podemos esperar que a aplicação rode usando
esse novo compilador?

15s = qt_I x CPI x T_CLK

Tn = .6 x qt_I x (CPI x 1.1) x T_CLK
Tn = .6 x 1.1 x 15 = 9.9s

Componetes de desempenho                        Unidades de medida
> Tempo de execução de um programa pela CPU     > Programa por segundo
> Contagem de instruções                        > Instruções por programa
> Instruções por ciclo de Clock (CPI)           > média de ciclos de clock por instruções 
> Tempo de ciclo de clock                       > Segundos por ciclo de clock
